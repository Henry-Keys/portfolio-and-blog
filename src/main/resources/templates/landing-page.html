Projects I helped delivering

BOX - Box Office Experience

BOX, developed by NEXT BSS, it's an application where users can buy tickets for parties, movie theaters, and pay for them completely online. BOX is the first platform that provided the truly first way to pay online end-to-end in Angola.

I was fortunate to work with the BOX mobile team, coding features such as ticket transfer, user profile management, connection and sharing to social networks, an Android/iOS Flutter plugin for push notifications, and many other features. I was also responsible for setting up the deployment process to AppStore, as well as fixing issues related to the iOS side of the code.

After this, I also worked with the backend team, developing features related to push notification delivery, ticket authenticity validation with QR, test coverage increase in order to allow refactoring and code improvement.
